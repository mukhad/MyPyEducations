Общие комманды.

:q - выйти	:q! - выйти без сохранения изменений
:w - сохранить  :r text.txt - прочитать
:e file_name - загрузить
:tab**** - работа с несколькими вкладками
:buffers - список всех буфферов

. - повтор предъидущей комманды
u (undo)
ctrl+r (redo)

:help copy + "ctrl+D" - вызов справки по "copy"
:set number / :set nonumber
:set whichwrap = b,s,<,>,[,] - установить переход с начала строки в конец предъидущей и наоборот при нажатии стрелок

:set shiftwidth = n
:set autoindent - отступ как у предъидущей строки
:set smartindent - с учетом скобок
:set cindent - smart для C/C++

/ "строка" - поиск от курсора
? "строка" - поиск с начала документа
n / shift+n - перемещение по найденным строкам
:set ignorecase - поиск без учета регистра (одноразово /c )
:set incsearch - инкрементный поиск
:nohlsearch - снять выделение с найденных строк

Перемещение курсора по тексту.

h - влево	j - вниз	k - вверх	l - вправо
shift+h / shift+l - начало/конец текущей строки
b / w	- 1 слово влево/вправо
e	- конец слова
ge	- начало слова
[ / ]	- начало текущего/следующего предложения
{ / }	- начало текущего/следующего абзаца

^ / $	- начало/конец строки
%	- к закрыващей скобке () {} []
gg / shift+g	- начало/конец файла
	ctrl+g	- "где я"



Форматирование текста.

i - insert	a - append
dw - удаление слова	dd - удаление строки
p (put) - вставляет последний удаленный элемент
shift+p - вставляет после курсора
cw 	- замена слова
v - visual - выделение символов
shift+v	-   выделение строк
ctrl+v	-   выделение блока текста 
	aw - +слово; as - +предложение; ap - +абзац;
	ab - +();    a+shift+b - +{};
	o - меняет положение курсора между 
	    началом/концом выделения 
x - удалить выделенный текст ( в буфер )
y - копировать выделенный текст в буфер обмена
yy - копирует строку
y+"комманда" - копирует по "движению" курсора
	y} - до конца абзаца
:%s/ABC/EFG/g - заменяет ABC на EFG без подтверждения
:%s/ABC/EFG/gc - заменяет ABC на EFG c подтверждением
	o - прервать замену; a - заменить все до конца.
	(больше про замену :help substitute)
